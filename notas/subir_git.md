1️⃣ Abrir el proyecto

Abrir la carpeta del proyecto en Visual Studio Code.

2️⃣ Inicializar Git

Abrir la terminal y ejecutar:

# git init

3️⃣ Agregar los archivos del proyecto
# it add .

4️⃣ Crear el primer commit
# git commit -m "primer commit"


5️⃣ Conectar el repositorio remoto
Copiar la URL del repositorio de GitHub.

Ejemplo:

# git remote add origin https://github.com/usuario/proyecto.git
6️⃣ Definir la rama principal
# git branch -M main

7️⃣ Subir el proyecto
# git push -u origin main
